Constructs a new Vector3 in a particular direction using a [[lang.luau.types.userdata.enum.normalid]].
```Lua
Vector3.fromNormalId(NormalId normalId)
```
## Example
```Lua
local myVector3 = Vector3.fromNormalId(Enum.NormalId.Front)
```
